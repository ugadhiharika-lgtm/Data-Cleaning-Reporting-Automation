# Data Cleaning & Reporting Automation

## Project Overview

This project automates data cleaning and reporting workflows using Python.

The system processes raw employee data, removes duplicate records, handles missing values, generates summary reports, and creates visual charts automatically.

## Features

- Remove duplicate records
- Handle missing values
- Standardize dataset
- Generate automated summary reports
- Create visual charts using Matplotlib
- Export cleaned data to CSV

## Technologies Used

- Python
- Pandas
- Matplotlib

## Project Files

### data_cleaning.py
Main Python script for data cleaning and report generation.

### cleaned_data.csv
Contains cleaned dataset after processing.

### summary_report.csv
Contains statistical summary of the dataset.

### department_chart.png
Visualization showing employee count by department.

## How to Run

### Install Dependencies

```bash
pip install pandas matplotlib
```

### Run Program

```bash
python data_cleaning.py
```

## Output

The script generates:

- cleaned_data.csv
- summary_report.csv
- department_chart.png
